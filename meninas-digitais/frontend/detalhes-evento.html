Workshop de Python Básico para Iniciantes

Workshop 15 vagas

Este workshop é perfeito para quem quer dar os primeiros passos no mundo da programação com Python! Abordaremos os conceitos fundamentais da linguagem, estruturas de dados básicas, lógica de programação e como escrever seu primeiro código funcional. Não é necessário nenhum conhecimento prévio em programação.

Ao final do workshop, você terá uma base sólida para continuar seus estudos em Python e estará apta a criar pequenos scripts e resolver problemas simples de codificação. Prepare-se para aprender de forma prática e divertida!
